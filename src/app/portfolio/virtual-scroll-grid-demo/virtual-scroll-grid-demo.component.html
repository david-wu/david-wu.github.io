Virtual Scroll Grid Demo

VirtualScrollGrid is useful for displaying a large number of tiles. Tile elements are not mounted onto the DOM until they are scrolled into the viewport. This allows for lazy loading of resources and improves render times.

The width and height of the tiles automatically adjusted to fill the virtualScrollGridComponent's area. This can be used to load lower resolution images when higher resolution images are not required.
